# Little Lemon Restaurant

A modern, responsive web application for the Little Lemon restaurant, featuring a table reservation system.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Table Reservation**: A functional booking form with validation.
- **Unit Tests**: Comprehensive testing for components and validation logic.
- **Accessibility**: Semantic HTML and ARIA attributes for screen reader support.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Csanad111/little-lemon.git
   ```
2. Navigate to the project directory:
   ```bash
   cd little-lemon
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

Start the development server:
```bash
npm start
```
The application will open in your browser at `http://localhost:3000`.

### Running Tests

Run the test suite:
```bash
npm test
```

## Project Structure

- `src/components`: React components (Header, Footer, BookingForm, etc.)
- `src/assets`: Images and static assets
- `src/App.css`: Global styles and design system

## Technologies Used

- React
- CSS3 (Grid/Flexbox)
- Jest & React Testing Library
